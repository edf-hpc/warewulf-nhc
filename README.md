warewulf-nhc
============

Warewulf Node Health Check Debian package, based on http://go.lbl.gov/nhc


How to build this package
~~~~~~~~~~~~~~~~~~~~~~~~~

1. git clone https://github.com/edf-hpc/warewulf-nhc warewulf-nhc
2. sudo apt-get install git-buildpackage
3. cd warewulf-nhc
4. git-buildpackage
